# :hammer: 환경설정 및 실행 방법

```
git clone https://github.com/wowns9270/last_lol.git

cd last_lol

npm i

-> 리그오브레전드의 api를 발급 받아야 합니다. (과정 생략)

발급받은 api 키를 사용하기 위해 .env 파일을 만들고
api_key={~~~}  {}는 생략

-> package.json 부분  script . start 부분 "nodemon app"으로 수정
-> 로컬에서 실행가능
```
